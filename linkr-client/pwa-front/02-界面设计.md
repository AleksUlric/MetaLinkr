# MetaLinkr PWA 界面设计文档

## 设计风格指南

### 整体设计理念
MetaLinkr 采用 Soul 风格的社交应用设计，结合现代 PWA 技术，打造沉浸式的社交体验。设计遵循以下核心理念：

- **Soul 风格**: 神秘、浪漫、个性化的视觉风格
- **星球概念**: 以星球为载体的社交空间设计
- **现代简约**: 简洁的界面，突出内容本身
- **情感连接**: 通过视觉设计促进用户情感交流

### 组件设计原则
- **一致性**: 统一的组件样式和交互模式
- **简洁性**: 清晰的信息层次，避免视觉混乱
- **响应性**: 适配不同屏幕尺寸和设备
- **可访问性**: 支持无障碍访问和操作
- **情感化**: 通过视觉元素传达情感和氛围

### 组件库规范
- **基础组件**: 按钮、输入框、卡片、列表等
- **复合组件**: 导航栏、搜索框、用户卡片等
- **业务组件**: 匹配卡片、直播卡片、商品卡片等
- **Soul 特色组件**: 星球卡片、灵魂匹配、瞬间发布等

## Soul风格设计系统

### 色彩方案
- **主色调**: #ff77c6 (Soul粉) - 浪漫、温暖
- **辅助色**: #7877c6 (Soul紫) - 神秘、优雅
- **背景色**: rgba(26, 26, 26, 0.9) (深色背景) - 沉浸感
- **文字色**: #ffffff (白色) - 对比度强
- **强调色**: #ff6b6b (红色) - 重要提示

### 渐变效果
- **主要渐变**: linear-gradient(135deg, #ff77c6 0%, #7877c6 100%)
- **悬停渐变**: linear-gradient(135deg, #ff6bb3 0%, #6b6bb3 100%)
- **背景渐变**: linear-gradient(135deg, #faf7ff 0%, #f3f0ff 100%)

### 圆角规范
- **小圆角**: 8px - 按钮、输入框
- **中圆角**: 12px - 卡片、弹窗
- **大圆角**: 20px - 大卡片、模态框
- **圆形**: 50% - 头像、图标容器

### 阴影效果
- **轻微阴影**: 0 2px 8px rgba(0, 0, 0, 0.1)
- **中等阴影**: 0 4px 16px rgba(0, 0, 0, 0.15)
- **深度阴影**: 0 8px 32px rgba(0, 0, 0, 0.2)

- **触摸友好**: 按钮最小44px，间距充足

- **多列布局**: 充分利用屏幕宽度
- **分屏支持**: 适配多任务模式
- **键盘适配**: 输入时自动调整布局
- **鼠标支持**: 悬停效果和右键菜单

### 桌面端适配
- **固定宽度**: 最大宽度限制，居中显示
- **悬停效果**: 丰富的鼠标悬停交互

## 页面结构设计

### 主要页面布局
1. **星球页面 (PlanetHome)**: Soul风格主页面，用户星球信息展示
2. **广场页面 (SquareHome)**: 用户动态浏览和互动
3. **聊天页面 (ChatList)**: 聊天列表和实时聊天功能
4. **个人页面 (UserProfile)**: 用户信息、成就、设置

### 页面路由结构
```
/ (重定向到 /login)
├── /login - 登录页面
│   ├── 手机号密码登录
│   ├── 测试登录功能
│   ├── 第三方登录（微信、QQ、微博）
│   └── 注册链接
├── /register - 注册页面
├── /forgot-password - 忘记密码
└── /app - 主应用布局
    ├── /app/planet - 星球页面
    ├── /app/square - 广场页面
    ├── /app/chat - 聊天页面
    └── /app/profile - 个人页面
```

### 功能页面
- **匹配相关**: /match/soul, /match/voice, /match/swipe
- **聊天相关**: /chat/:id, /chat-group/:id
- **动态相关**: /post/:id, /post/create
- **用户相关**: /user/:id, /profile/edit
- **设置相关**: /settings, /settings/privacy
- **其他功能**: /live, /shop, /game, /gift

## 无障碍设计

### 视觉无障碍
- **对比度**: 文字与背景对比度≥4.5:1
- **字体大小**: 最小14px，支持系统字体缩放
- **色彩依赖**: 不仅依赖颜色传达信息
- **焦点指示**: 清晰的焦点状态指示

### 操作无障碍
- **键盘导航**: 所有功能支持键盘操作
- **语音支持**: 支持屏幕阅读器
- **手势替代**: 提供点击替代手势操作
- **错误提示**: 清晰的错误信息和纠正建议

## 组件状态设计

### 默认状态
- **正常显示**: 组件的基本外观
- **可交互**: 明确的可点击区域
- **信息完整**: 显示必要的文字和图标

### 悬停状态
- **视觉反馈**: 颜色变化、阴影增强
- **状态提示**: 显示操作提示信息
- **动画过渡**: 平滑的状态切换动画

### 激活状态
- **按下效果**: 按钮按下时的视觉反馈
- **状态保持**: 选中状态的持续显示
- **操作确认**: 明确的操作执行反馈

### 禁用状态
- **视觉降级**: 降低对比度和饱和度
- **交互禁用**: 移除所有交互功能
- **状态说明**: 显示禁用原因或条件

### 加载状态
- **加载指示**: 旋转动画或进度条
- **内容保护**: 防止加载过程中的误操作
- **状态提示**: 显示加载进度或预计时间

### 错误状态
- **错误提示**: 清晰的错误信息显示
- **视觉警示**: 使用红色等警示色彩
- **恢复机制**: 提供错误恢复的操作选项

## 组件交互规范

### 点击交互
- **响应时间**: 点击后100ms内给出反馈
- **视觉反馈**: 按钮缩放、颜色变化等
- **状态保持**: 选中状态持续显示
- **防误触**: 重要操作需要二次确认

### 滑动交互
- **滑动方向**: 支持上下左右四个方向
- **滑动距离**: 最小滑动距离20px
- **滑动速度**: 根据滑动速度调整动画
- **边界处理**: 滑动到边界时的视觉反馈

### 长按交互
- **触发时间**: 长按500ms触发
- **视觉反馈**: 长按过程中的视觉提示
- **功能菜单**: 显示上下文菜单或操作选项
- **取消机制**: 支持滑动取消长按操作

### 双击交互
- **触发间隔**: 双击间隔300ms内
- **功能区分**: 与单击功能明确区分
- **视觉反馈**: 双击时的特殊动画效果
- **适用范围**: 主要用于图片放大、视频播放等
